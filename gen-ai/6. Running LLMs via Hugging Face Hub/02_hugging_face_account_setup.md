# Hugging Face Account Setup

## Objective

Create and verify a Hugging Face account so you can access models, Spaces, datasets, and gated model workflows.

---

## Signup Flow

1. Open `https://huggingface.co`
2. Click **Sign Up**
3. Complete the human verification challenge
4. Enter email + password
5. Choose a unique username
6. Optionally fill full name and profile details
7. Create account

---

## Email Verification (Required)

After signup, confirm your email from the inbox link.

Why this matters:

- activates account fully
- avoids access limits on some model pages and features
- ensures recovery and security workflows work correctly

---

## Optional Profile Setup

You can customize profile basics right after account creation:

- avatar
- display name
- bio/organization fields (optional)

These are not mandatory for basic model usage.

---

## First Dashboard Tour

After login, key areas to explore:

| Area | Purpose |
|---|---|
| Models | Discover and use open-source models |
| Spaces | Run hosted demos/playgrounds |
| Datasets | Find datasets for training/fine-tuning |
| Community/Docs | Learn usage patterns and best practices |

---

## Key Takeaways

- Account setup is simple: signup, verify email, choose username.
- Email confirmation is the important completion step.
- Once logged in, Models/Spaces/Datasets are the main working tabs.
- This account is the base requirement for the next steps: model access and local runs.
