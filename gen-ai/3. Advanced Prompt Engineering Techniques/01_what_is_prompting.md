# What Is Prompting?

## Why This Section Matters

Prompting is one of the highest-leverage skills in agentic AI.

The same model can produce very different results depending on how you ask.
With better prompts, you can often improve output quality and accuracy by a large margin without changing the model.

---

## What Is a Prompt?

A prompt is the instruction you give to an LLM.

It can include:

- task description
- context
- constraints
- examples
- expected output format

A clear prompt guides the model toward useful, reliable responses.

## Key Prompting Types Covered

- Zero-shot prompting
- Few-shot prompting
- Chain-of-thought prompting

Each has a different purpose:

- zero-shot for direct instruction without examples
- few-shot for pattern learning from examples
- chain-of-thought for structured reasoning workflows
