# Model Context Protocol (MCP) – Section Overview

## What is MCP?

**MCP (Model Context Protocol)** is a protocol that standardizes the way you provide context to AI models.

It is relatively new in the AI world but has been heavily adopted by large companies and enterprises.

## What This Section Covers

- What MCP is and what problem it solves
- How it standardizes the way AI agents interact with tools and external context
- How AI agents can leverage MCP to make tool calls more standardized
